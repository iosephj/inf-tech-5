### 1. Algunos conceptos web básicos


Una **página web** es un documento digital que puede contener texto, imágenes, videos, enlaces y otros elementos interactivos. Por ejemplo: un artículo de noticias, una tienda en línea, etc. Las páginas web se visualizan mediante un programa llamado **navegador**.

Un **sitio web** es un conjunto de páginas web relacionadas entre sí. Por ejemplo un sitio web de noticias contiene muchas páginas con distintas noticias.

**¿Qué es Internet?** Es una red mundial de computadoras conectadas entre sí. Las páginas web utilizan Internet para llegar desde una computadora hasta otra.


<br>

Cuando visitas una página web intervienen dos elementos importantes:

**Navegador:** Es el programa que utilizamos para ver páginas web. Ejemplos: Google Chrome, Mozilla Firefox, Microsoft Edge. El navegador solicita la página y la muestra en pantalla.

**Servidor:** Es una computadora que almacena páginas web y las envía cuando alguien las solicita. Por ejemplo:

1. Escribes una dirección web.
2. El navegador pide la página al servidor.
3. El servidor la envía.
4. El navegador la muestra.


<br>

### 3. ¿Cómo está construida una página web?

La mayoría de las páginas web utilizan tres tecnologías principales:

| Tecnología | Función                         |
| ---------- | ------------------------------- |
| HTML       | Estructura y contenido          |
| CSS        | Apariencia y diseño             |
| JavaScript | Interactividad y comportamiento |

#### HTML

HTML indica qué elementos contiene una página.

Ejemplos:

* Títulos
* Párrafos
* Imágenes
* Tablas
* Enlaces

Podemos imaginar HTML como el esqueleto de una casa.

#### CSS

Estas siglas provienen de "Cascading Style Sheets" (Hojas de Estilo en Cascada). CSS controla la apariencia.

Permite modificar:

* Colores
* Tamaños
* Bordes
* Posición de los elementos

Podemos imaginar CSS como la pintura y decoración de la casa.

#### JavaScript

JavaScript permite que la página responda a acciones del usuario.

Por ejemplo:

* Mostrar mensajes.
* Realizar cálculos.
* Validar formularios.
* Cambiar contenido dinámicamente.

Podemos imaginar JavaScript como los mecanismos que hacen funcionar la casa.
