🧪 Diagnóstico inicial — Informática

Si no conocés el lenguaje, intentá deducir qué hace el código. No se evalúa memoria sino razonamiento.

1. ¿Qué imprime este código?

x = 5
if x > 3:
    print("A")
else:
    print("B")
  1. A
  2. B
  3. Error
  4. Nada

2. ¿Qué hace este código?

for i in range(3):
    print(i)
  1. Imprime 1, 2, 3
  2. Imprime 0, 1, 2
  3. Imprime 0, 1, 2, 3
  4. Da error

3. ¿Cuál es el resultado?

a = 10
b = 3
print(a // b)
  1. 3.33
  2. 3
  3. 4
  4. Error

4. ¿Qué representa mejor una variable?

  1. Un número fijo
  2. Un lugar donde se guarda un valor
  3. Una operación matemática
  4. Un tipo de dato

5. ¿Qué es un algoritmo?

  1. Un lenguaje de programación
  2. Una computadora
  3. Una serie de pasos para resolver un problema
  4. Un programa ya hecho

6. Si una fórmula en una planilla da mal, lo más probable es que:

  1. La computadora esté rota
  2. El programa esté mal diseñado
  3. El usuario cometió un error en los datos o fórmula
  4. Es imposible saber

7. ¿Cuál de estas es una condición válida?

  1. x = 5
  2. x == 5
  3. 5 x
  4. =x5

8. ¿Qué hace un bucle (loop)?

  1. Guarda datos
  2. Repite acciones
  3. Compara valores
  4. Muestra resultados
